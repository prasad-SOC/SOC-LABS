# SOC Labs

## Overview

SOC Labs is a hands-on cybersecurity practice repository focused on building practical skills required for Security Operations Center (SOC) roles.

This project contains structured labs using commonly used defensive security tools such as Nmap, tcpdump, Wireshark, Splunk, CrowdStrike Falcon, Suricata, and Linux.

Each section includes multiple labs designed to simulate tasks typically performed by SOC analysts, including network scanning, packet inspection, log investigation, intrusion detection, and system analysis.

---

## Objectives

* Practice real-world SOC investigation workflows
* Develop hands-on experience with core security tools
* Understand network traffic and system activity analysis
* Build a documented cybersecurity lab portfolio

---

## Tools Covered

* Nmap – Network discovery and port scanning
* tcpdump – Command-line packet capture and analysis
* Wireshark – Network packet analysis
* Splunk – Log analysis and security monitoring
* CrowdStrike Falcon – Endpoint detection and response
* Suricata – Network intrusion detection system
* Linux – System navigation, file analysis, and log investigation

---

## Lab Methodology

Each lab includes:

* Objective
* Required commands or queries
* Step-by-step procedure
* Expected output
* Screenshots for verification
---

## Purpose

The purpose of this repository is to document practical cybersecurity learning through repeatable labs and real investigation workflows.

This repository serves as a personal SOC learning portfolio demonstrating hands-on experience with security monitoring and analysis tools.
